# EasyWord

#### 介绍
基于Apache POI的API，实现更加人性化简易化的word生成和数据插入类功能




#### 目标计划
1.  [√] word文档的文本和表格占位符替换功能
2.  [  ] word文档图标绘制
    - [  ] 折线图
    - [  ] 饼图
    - [  ] 柱状图
    - [  ] 散点图


#### 使用说明
详见 [easy-word-demo](https://gitee.com/yuuma/easy-word-demo/src/master/src/main/java/com/yuuma/EasyWordDemo.java)
